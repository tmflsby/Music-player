## 🚀 如何运行

### ✨开发过程

```
# 克隆
git clone https://github.com/tmflsby/Music-player.git
```

```
# 打开项目目录
cd Music-player
cd music-Player
```

```
# 安装依赖
npm install
```

```
# 开启本地服务运行项目
npm run serve
```
